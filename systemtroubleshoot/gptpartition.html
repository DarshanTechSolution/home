How to solve - Windows cannot be installed to this Disk. The selected disk is of the GPT partition style.

While installing OS the error Windows cannot be installed to this Disk. The selected disk is of the GPT partition style will occured. This page clearly explains step by step how to resolve this error.

Follow the below steps to clear the error

1. Must backup your data before Proceed

2. Boot from windows installation disk

3. Choose the language, and then click Next

4. Press Shift + F10 . Now command prompt will open.

Type the following commands in command prompt

  • diskpart
  • list disk
  • select disk 0
  • clean
  • exit

    • Now restart the system and continue the OS installation process.
